# Currency Converter

A web application currency converter is a simple web application tool that allows users to convert the value of PLN (Polish Zloty) into 3 other curency: EUR (Euro), USD (United States Dollar) and CAD (Canadian Dollar). Users input the amount in PLN they wish to convert along with the source and target currencies, and the application displays the converted amount based on current exchange rates from The National Bank of Poland (20-FEB-24).
